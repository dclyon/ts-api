ran curl command whle hosting via Flask on port 5000. this grants token:

curl -X POST http://localhost:5000/login -H "Content-Type: application/json" -d '{"username":"user1", "password":"password1"}'

(add -k if self signed cert err)

Breakdown:
    -X POST: Specifies that this is a POST request.
    http://[your-api-url]/login: The URL to your login route.
    -H "Content-Type: application/json": Sets the Content-Type header to application/json to indicate that the request body is a JSON object.
    -d '{"username":"yourUsername", "password":"yourPassword"}': Provides the request data (i.e., the username and password) as a JSON object.
